Para criar um ambiente para trabalhar com o NFE basta executar o comando abaixo:
```
  docker-compose up -d
```

Para parar o container basta digitar:
```
  docker-compose down
```

## Tecnologias
* [Docker](https://www.docker.com/)
* [PHP 7.x](http://php.net/)
* [Lumen 5.8](https://lumen.laravel.com/)
